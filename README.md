# Intro to Copilot Chat

## Usecase 1: Email Drafting

### Bad Prompt

```
Write an email to a client.
```

* No context
* No audience details
* No tone specified
* No expected outcome

### Good Prompt

```
Write a professional email to a client informing them that our project timeline has been moved by two weeks.
```

* Clear task
* Identifies purpose
* Specifies tone

### Excellent Prompt

```
I’m a project manager at a software company. Write a professional but reassuring email to our client informing them that the mobile app launch has been delayed by two weeks due to additional QA testing. Emphasize that the delay will improve reliability and user experience. Keep the email concise, use a confident tone, and end with an invitation for questions.
```

* Includes context
* Defines the goal
* Sets tone and expectations
* Gives business reasoning
* Specifies format and style

## Usecase 2: Excel Data Analysis

### Bad Prompt

```
Analyze this spreadsheet.
```

* Too vague
* No desired output
* No focus area

### Good Prompt

```
Analyze this sales spreadsheet and identify the top-performing products.
```

* Defines the task
* Identifies what to analyze

### Excellent Prompt

```
Analyze this quarterly sales spreadsheet and identify the top 5 performing products based on total revenue. Create a summary table and highlight any sales trends by region. Use a business-friendly tone and include 3 actionable recommendations for improving low-performing regions.
```

* Specifies metrics
* Requests format
* Adds business context
* Defines expected output

## Usecase 3: PowerPoint Presentation

### Bad Prompt

```
Make slides about cybersecurity.
```

* No audience
* No depth
* No structure
* No presentation goal

### Good Prompt

```
Create a PowerPoint presentation about cybersecurity best practices for employees.
```

* Defines topic
* Identifies audience

### Excellent Prompt

```
Create a 10-slide PowerPoint presentation for non-technical office employees about cybersecurity best practices. Include topics such as phishing, password security, MFA, and safe browsing habits. Use concise bullet points, simple language, and suggest visuals or icons for each slide. End with a short security checklist employees can follow daily.
```

* Defines audience
* Specifies structure
* Sets expectations
* Provides presentation style guidance

## Usecase 4: Office Automation / Workflow Automation

### Bad Prompt

```
Automate invoices.
```

* No process details
* No tools specified
* No expected output

### Good Prompt

```
Create an automation workflow for generating invoices from Excel data.
```

* Defines task
* Mentions data source

### Excellent Prompt

```
I work in finance operations and want to automate monthly invoice generation using Microsoft 365 tools. Create a workflow using Excel, Word templates, and Power Automate that:

* Reads customer billing data from Excel
* Generates individualized invoices using a Word template
* Saves invoices as PDFs
* Emails invoices to customers automatically through Outlook
* Creates a tracking log in Excel

Include step-by-step implementation instructions and recommended naming conventions for files and folders.
```

* Complete business context
* Clear workflow goals
* Specific tools
* Defined deliverables
* Actionable output expectations

```

Executive Summary
Revenue and regional sales performance
Customer feedback trends
Operational risks and challenges
Strategic initiatives and opportunities
Recommendations for Q3 2026

Use clear section headings
Include bullet points for key findings
Present sales information in a table
Use a formal executive tone
Keep the report concise (approximately 1–2 pages)
End with 3 actionable recommendations for management
```

* Provides business context
* Defines the exact structure
* Specifies formatting expectations
* Identifies the audience
* Guides Copilot toward a high-quality executive-style output
